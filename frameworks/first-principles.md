# First Principles

**Phase:** Define · **Source:** https://untools.co/first-principles

## Entry Predicate
`always_run`

## Inputs
- `intake.problem_refined`

## Method
1. List the **assumptions** baked into the current problem framing. (Use research evidence to expose unstated ones.)
2. For each assumption, ask: is this physically/logically necessary, or just convention?
3. Strip to **atomic truths** — what cannot be reduced further.
4. Rebuild a solution from atoms only, ignoring conventional approaches.

## Output Schema
```
ASSUMPTIONS (numbered)
  1. <assumption> — [necessary | convention | unknown]
  2. ...

ATOMIC TRUTHS
  - <truth>
  - <truth>

REBUILT FRAMING
  <new problem statement using only atoms>
```

## Decision Hook
Compare rebuilt framing against original. If they diverge significantly → flag for decide-phase Hard Choice (the conventional path may be wrong).

## What This Means For The Decision
First principles surfaces solutions invisible inside the current frame. If atoms show a constraint isn't real, the cheapest decision may be one nobody else considered.
